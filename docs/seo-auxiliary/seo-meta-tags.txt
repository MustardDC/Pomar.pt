===== PRÓXIMOS PASSOS =====

Para integrar estes meta tags no Jekyll:

1. Abra cada ficheiro .md
2. Localize a secção front matter (entre ---)
3. Adicione ou atualize:

---
layout: especie
title: Maçã
description: [Copie description acima]
keywords: [Copie keywords acima]
og_title: [Copie og_title acima]
og_description: [Copie og_description acima]
---

4. Nos layouts (_layouts/especie.html, _layouts/pratica.html, default.html):
   Adicione:
   <meta name="description" content="{{ page.description }}">
   <meta name="keywords" content="{{ page.keywords }}">
   <meta property="og:title" content="{{ page.og_title }}">
   <meta property="og:description" content="{{ page.og_description }}">

5. Commit e push para GitHub Pages
