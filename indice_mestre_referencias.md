## GLOSSÁRIO TÉCNICO

### TERMOS FISIOLOGIA E ESTRUTURA

| Termo | Definição | Contexto Uso |
|------|-----------|------------|
| **Gema** | Estrutura pequena contém crescimento potencial (flor ou ramo) | Enxertos, poda |
| **Verdasca** | Raminhos curtos (12-15cm) com gomos florais | Maçã, pera |
| **Gomo Floral** | Gema que produz flores/frutos | Identificação ramos produtivos |
| **Dormência** | Período repouso fisiológico inverno | Poda, enxertos, planejamento |
| **Indução Floral** | Transformação gema vegetativa → floral | 6-12 meses antes floração |
| **Incompatibilidade** | Incapacidade enxerto e porta-enxerto soldarem | Enxertos (causa falha) |
| **Compatibilidade** | Capacidade enxerto/porta-enxerto soldarem bem | Enxertos (sucesso) |
| **Climatérico** | Fruto amadurece pós-colheita (continua processo) | Maçã, pera, pessego |
| **Não-climatérico** | Fruto NÃO amadurece pós-colheita | Uva, morango, citros |
| **Fotossíntese** | Conversão luz em energia; cria açúcares | Crescimento árvore |
| **Translocação** | Movimento nutrientes na planta | Adubação, fisiologia |
| **Horas Frio** | Horas acumuladas <7°C no inverno | Selecção variedades |
| **Brix** | Medida açúcar no fruto (°Brix) | Índice maturidade |
| **Floema** | Tecido transporte açúcares (enxertia crítico) | Enxertos, compatibilidade |
| **Xilema** | Tecido transporte água, nutrientes | Estrutura árvore |

---

### TERMOS PRÁTICA AGRÍCOLA

| Termo | Definição | Uso Prático |
|------|-----------|-----------|
| **Desbaste** | Remoção flores/frutos em excesso | Melhorar qualidade fruto |
| **Desponta/Despontamento** | Corte ponta ramo | Estimular ramificação lateral |
| **Ladrão** | Ramo cresce verticalmente da base/tronco | Indesejável; remover |
| **Poda Sanitária** | Remover partes doentes/mortas | Prevenção doença |
| **Poda Seletiva** | Escolher especificamente ramos remover | Poda produção |
| **Tutor** | Suporte (estaca/arame) | Orientar crescimento |
| **Espalier** | Forma árvore alongada horizontalmente | Videira, citrinos |
| **Cordon** | Videira com um ou dois eixos principais | Videira forma |
| **Armazém Vivo** | Deixar fruto na árvore armazenagem | Citrinos meses |
| **Fertirrigação** | Adubo dissolvido em água rega | Aplicação eficiente |
| **Porta-enxerto** | Árvore base (raízes) para enxertia | Estrutura, vigor |
| **Enxerto** | Ramo/gema inserida em porta-enxerto | Variedade desejada |
| **Brotação** | Saída folhas/ramos primavera | Ciclo vegetativo |
| **Floração** | Saída flores | Reprodução árvore |
| **Frutificação** | Crescimento fruto após floração | Produção |
| **Colheita** | Recolha frutos maduros | Produção final |
| **Poda Verde** | Poda durante vegetação (verão) | Ventilação, luz |

---

### TERMOS DOENÇAS/PRAGAS

| Termo | Significado | Contexto |
|------|-----------|---------|
| **Fungicida** | Produto mata/inibe fungos | Tratamento doenças fúngicas |
| **Inseticida** | Produto mata/controla insetos | Controlo pragas |
| **Bactericida** | Produto contra bactérias | Cancro bacteriano |
| **Sistémico** | Produto entra planta, circula internamente | Eficaz infecção instalada |
| **Contacto** | Produto mata por contacto superficial | Ação rápida, localizada |
| **Bioacaricida** | Produto natural contra ácaros | Controlo ecológico |
| **Prevenção** | Evitar problema (melhor que curar) | Estratégia primária |
| **Controlo Integrado** | Múltiplas estratégias (não só químico) | Manejo sustentável |
| **Inimigos Naturais** | Predadores/parasitóides de pragas | Controlo ecológico |
| **Incidência** | Percentagem plantas afetadas | Medida gravidade |
| **Severidade** | Intensidade sintomas planta afetada | Medida gravidade |
| **Carência** | Dias após pulverização até colheita | Segurança alimentar |
| **Resistência** | Capacidade árvore resistir praga/doença | Genética variedade |
